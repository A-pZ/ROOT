'Struts2-Thymeleaf Plugin' is enable to use result type 'thymeleaf' and 'thymeleaf-spring'. I recommend 'thymeleaf-spring' which could display field error and fields provided diarect 'sth' , SpringBeans behavior for read from static resources.
